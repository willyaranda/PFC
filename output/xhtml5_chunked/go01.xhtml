Glosario de términos

A

API

Application Program Interface. Conjunto de funciones y procedimientos que ofrece cierta biblioteca o librería para ser utilizado por otro software como una capa de abstracción.

B

BTS

Base Transceiver Station. Definida para comunicaciones radio, es una instalación fija para la comunicación media, baja o alta bidireccional. Se usa para comunicar con una o más radios móviles o teléfonos celulares. Dispone de equipos transmisores y receptores de radio, y en UMTS son los que realizan el enlace con el usuario que efectúa o recibe la llamada con un teléfono móvil.

D

DLNA

Digital Living Network Alliance. Es una asociación de fabricantes de electrónica e informática cuyo principal cometido es definir directrices de interoperabilidad que permitan compartir medios digitales entre dispositivos de consumo como ordenadores, impresoras, teléfonos móviles y otros dispositivos multimedia.

DOM

Document Object Model. Es un API que proporciona un conjunto estándar de objetos para representar documentos HTML y XML, un modelo estándar sobre ´como cpueden combinarse dichos objetos y una interfaz estándar para acceder a ellos y manipularlos.

F

Flood

Inundación. Referido a las redes, suele indicar que hay tal cantidad de paquetes, que se saturan y baja considerablemente la calidad del servicio.

I

IP

Etiqueta numérica que identifica, de manera lógica y jerárquica, a un interfaz de un dispositivo dentro de una red que utilice el protocolo IP, que corresponde al nivel de red del modelo OSI. Hay dos tipos, IPv4 e IPv6, cuya diferencia más percibible es el rango de direccionamiento expuesto por cada una.

J

JSON

JavaScript Object Notation[39]. Formato ligero para el intercambio de datos. Es fácil de escribir y fácilmente entendible, siendo también rápido y sencillo de parsear.

L

LAN

Local Area Network. Red de área local. Interconexión de una o varios ordenadores y dispositivos. Su principal cometido es la creación de redes privadas.

M

MCC

Mobile Country Code. Código numérico usado para identificar el país de una red móvil. Está formado por 3 cifras suele ir unido a MNC.

MNC

Mobile Network Code. Código numérico que identifica un operador móvil. Suele estar formado por 2 cifras y va unido al MCC.

MD5

Algoritmo de reducción criptográfico de 128 bits ampliamente usado. Suele representarse como un número de 32 dígitos hexadecimal.

MongoDB

Base de datos no relacional que guarda sus datos en un formato BSON, similar a JSON, y cuyo lenguaje de consulta es JavaScript, teniendo funciones de reducción, mapeo…

N

NAT

Network Address Translation. Mecanismo usado por routers IP para intercambiar paquetes entre dos redes que asignan mutuamente direcciones incompatibles. Consiste en convertir, en tiempo real, las direcciones utilizadas en los paquetes transportados.

Notification Server

Servidor de notificaciones. Es el conjunto general en la parte centralizada, está compuesto de diferentes partes, como el NS_AS, NS_UA_WS…

Node.js

Plataforma de desarrollo en la parte servidora escrita principalmente en JavaScript. Permite una programación rápida, basada en eventos y asíncrona.

P

PID

Process IDentifier. Identificador del proceso. Suele ser un número entero usado por el kernel de algunos sistemas operativos para identificarlo de forma unívoca.

R

RabbitMQ

Sistema de mensajería escrito en Erlang, muy rápido, con muchas configuraciones para las colas y preferencias para alta disponibilidad y alta carga.

RSS

Really Simple Syndication. Formato X;L para indicar o compartir contenido en la web. Se suele utilizar para difundir información actualizada a usuarios que se han suscrito a la fuente de contenidos.

U

User Agent

Agente de usuario. Es el programa que ejecuta una instancia web. En este caso, podría ser el propio sistema operativo Firefox OS o el navegador Firefox.

UUID

Universally Unique IDentifier. Es un identificador estándar usado en el edesarrollo de software y estandarizado. Se suele utilizar para crear identificadores únicos por diferentes dispositivos o módulos, con la cierta confianza de que no habrá otro igual (debido al amplio dominio).

V

VoIP

Voz sobre IP. Grupo de recursos que hacen posible que la señal de voz viaje a través de internet empleando un protocolo IP. La señal de voz, se envía en forma digital, en contraste con el envío analógico de la mayoría de las redes de telefonía.

W

WAN

Wide Area Network. Red de área amplia, es un tipo de red de ordenadores capaz de cubrir grandes distancias, proveyendo de servicio a un país o continente.

WebApp

Aplicación web. Programa instalado en el dispositivo, basado en tecnologías web.

WebSocket

Tecnología que proporciona un canal de comunicación bidireccional y full-duplex sobre un único socket TCP. Está diseñada para ser implementada en navegadores y servidores web.

X

XMPP

eXtensible Messaging and Presence Protocol. Procolo abierto y extensible basado en XML, originalmente ideado para mensajería instantánea.
